# SleepyLights IR Remote Control Codes
Decoded IR remote control codes for the SleepyLights night lights (http://amzn.to/2l8FCjM).

## power button
Encoding  : NEC

Code      : FF609F (32 bits)

## white button
Encoding  : NEC

Code      : FF906F (32 bits)

## dark green button
Encoding  : NEC

Code      : FF9867 (32 bits)

## light green button
Encoding  : NEC

Code      : FFE817 (32 bits)

## yellow button
Encoding  : NEC

Code      : FF02FD (32 bits)

## dark blue button
Encoding  : NEC

Code      : FFD827 (32 bits)

## light blue button
Encoding  : NEC

Code      : FF48B7 (32 bits)

## cyan button
Encoding  : NEC

Code      : FF32CD (32 bits)

## pink button
Encoding  : NEC

Code      : FF8877 (32 bits)

## lavender button
Encoding  : NEC

Code      : FF6897 (32 bits)

## purple button

Code      : FF20DF (32 bits)

## red button

## light red button
Encoding  : NEC

Code      : FFB24D (32 bits)

## orange button

## sleep button
